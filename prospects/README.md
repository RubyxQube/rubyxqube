# Prospects

One folder per prospect. Each folder contains:

```
[business-slug]/
  PROFILE.md    ← researcher output (services, tone, content, aesthetic recommendation)
  demo/         ← built demo site files (deployed to Vercel preview URL)
```

## Status tracking
Pipeline status lives in Notion → Sales Pipeline → Prospects database.
Full reference: `docs/PROSPECT_DEMO_WORKFLOW.md`

## Naming convention
Use lowercase-hyphenated slug based on business name:
- `mountain-west-plumbing/`
- `treasure-valley-hvac/`
- `boise-tile-co/`
