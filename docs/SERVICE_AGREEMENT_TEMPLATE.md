- **Domain:** Client owns their domain. RubyxQube will assist with setup but will not register a domain in its own name on Client's behalf without written agreement.

- **Third-party services** (Google Analytics, AI chatbot API, contact form): RubyxQube will set these up. Ongoing API costs for Autopilot/Momentum are covered by the monthly retainer.
