# The Modern — Build Guidelines

> **Best for:** Real Estate · Architecture · Photography · Interior Design · Marketing Agency · Tech Startup · Creative Studio

---

## Core Philosophy

Precision and restraint. Every element earns its place. The dark canvas makes the content pop without needing color noise. This design says "we are the best option" without having to say it — the quality speaks for itself.

---

## Color System

| Role       | Default Value | Notes |
|------------|--------------|-------|
| Background | `#0a0a0a`    | True black — stark and premium |
| Navbar     | `#0a0a0a`    | Same as body — seamless |
| Hero bg    | `#0a0a0a`    | Continuous dark canvas |
| Accent     | `#e11d48`    | Ruby red — surgical use only |
| Text       | `#ffffff`    | Full white |
| Muted      | `#525252`    | Dark gray — secondary text |
| Cards      | `#171717`    | Barely lifted from black |

Client palette may override the accent — deep gold, electric blue, or neon green all work here. The key is ONE strong accent used sparingly.

---

## Typography

- **Headings:** Bold (700–800), large, commanding.
- **Hero H1:** 56–72px. Could be 2–3 words. White on black. Massive.
- **Accent type:** Use the accent color only for the key word or phrase you want to highlight. One or two words max.
- **Body:** 16–17px, slightly muted (rgba(255,255,255,0.75)). Reading-weight, not heavy.
- **Spacing:** Generous. Let things breathe. More margin than you think you need.

## Layout Structure (Top to Bottom)

### 1. Navbar
- Black or near-black background, seamless with the page
- Thin border bottom (1px, very subtle — rgba(255,255,255,0.06))
- Logo left, navigation links center or right, CTA button right
- CTA: outline style or filled accent — "Get in Touch" / "View Work"

### 2. Hero Section
- Full-height or very tall — this style earns the scroll
- Left-aligned OR centered — both work, but be deliberate
- **H1:** Short and confident. "We build things that matter." / "Boise's most awarded interior designer."
- **Subhead:** 1 sentence, muted text, smaller. No fluff.
- **CTA:** One primary. Accent color. Generous padding.
- Optional: Subtle looping video background, or a single striking full-bleed image

### 3. Featured Work / Portfolio Strip
- 3–4 portfolio pieces in a horizontal scroll or grid
- Each: client name, project category, full-bleed image
- Hover states: image slightly lightens, overlay text appears
- "See all work →" link at the end

### 4. Services / What We Do
- Clean grid (2–3 columns) OR a stacked list with generous spacing
- Service name in large text, 2-line description below in muted
- No icons necessary — or use thin line icons if client prefers
- Accent used sparingly: maybe just the service number (01 / 02 / 03) in accent color

### 5. About / Studio Story
- 2-column: large portrait or studio photo left, paragraph right
- OR: Single wide section with bold pull quote
- Keep copy tight: 3–4 sentences, no filler

### 6. Testimonials / Social Proof
- 2–3 quotes, large font, white text
- Attribution: name + company in muted text
- Can be a full-bleed section with dark background and accent left-border on quotes

### 7. Process (optional but effective)
- Numbered steps: 01 Discovery / 02 Strategy / 03 Build / 04 Launch
- Accent color on numbers
- 1-line description per step
- Shows sophistication and reduces sales friction

### 8. CTA Section
- Stark: dark background, big headline, one button
- "Ready to work together?" + "Let's Talk →"
- No form on this page — just the button to contact

### 9. Footer
- Very minimal: logo, nav links, social icons
- Copyright line, privacy policy link
- No clutter — this style dies by over-decorating the footer

## Mobile Rules
- Full-height hero scales to near-full on mobile
- Large headings shrink gracefully (use clamp() or responsive sizes)
- Portfolio grid becomes horizontal scroll or single column
- CTA always visible and tappable

---

## Content to Collect from Client
- [ ] Business name + owner/studio name
- [ ] 4–8 portfolio pieces (full-res images or project details)
- [ ] Services list with brief descriptions
- [ ] Process / how they work
- [ ] 2–3 client testimonials
- [ ] About/origin story (1–2 paragraphs)
- [ ] Social handles (especially Instagram for visual businesses)
- [ ] Brand direction preferences (minimalist? editorial? architectural?)

---

## Common Mistakes to Avoid
- Don't overuse the accent color — its power comes from scarcity
- Don't fill every section — white space (or black space) IS the design
- Don't add unnecessary animations — subtle fade-ins max; no bounce or slide
- Don't use more than 2 fonts — usually just one plus monospace for labels
- Don't add stock photos — this style demands real, high-quality imagery
